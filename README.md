# NotificationsAPI
simple API using node and expressJS .
> Notification's signup using Node.js/Express and the Mailchimp API. Fill in your [audience ID] and your [API Key] in app.js

## Quick Start

```bash
# Install dependencies
npm install

# Serve on localhost:3000
npm start
npm run dev (Nodemon)
```
